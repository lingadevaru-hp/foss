## Verification Notes

1. Solana RPC was the authoritative source for mint state, supply, transaction parsing, and current token-account balances.
2. IPFS was the authoritative source for the off-chain metadata JSON and image.
3. Orca API was the most useful live DEX source because GeckoTerminal returned null reserve/price fields and Solscan HTML was protected by Cloudflare in CLI fetches.
4. Coinbook provided strong corroborating evidence for the likely creation platform and the meaning of the top holder (`bondingCurve`).
